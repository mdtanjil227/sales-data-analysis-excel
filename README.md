# Sales Data Analysis (Excel)

## Project Overview
This project analyzes a retail sales dataset to understand business performance, product profitability, and regional sales trends. The analysis was conducted using Microsoft Excel with Pivot Tables and data visualization.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization

## Dataset
The dataset used is the Superstore Sales dataset, which contains information about orders, products, customers, regions, sales, and profits.

## Business Questions
- Which region generates the highest sales?
- Which product category produces the most revenue?
- Which category is the most profitable?
- How have sales changed over time?

## Key Insights
- The **West region** generated the highest sales.
- The **Technology category** produced the highest revenue and profit.
- **Furniture has high sales but very low profit margins**.
- Sales increased steadily from **2015 to 2017**.

## Dashboard
The Excel dashboard includes the following visualizations:
- Sales by Region
- Sales by Product Category
- Profit by Product Category
- Sales Trend (2014–2017)
